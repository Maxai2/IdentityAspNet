6. Подтверждение при регистрации
https://metanit.com/sharp/aspnet5/16.5.php
По ссылке описано подтверждение через почту.
Логика обычно такая:
+ пользователь вводит данные для регистрации
+ на почту отравляется письмо с ссылкой
+ пользователь идет на почту
+ переходит по ссылке и action из этого урла ставит галочку, что почта подтверждена
+ пользователь может залогиниться
+ пока не подтвердит почту, логин не проходит

Я не хочу, чтоб вы сейчас тратили время на то чтоб разобраться как отправлять почту, поэтому логика без этапа с почтой:
+ пользователь вводит данные для регистрации
+ на экране выводится ссылка
+ переходит по ссылке и action из этого урла ставит галочку, что почта подтверждена
+ пользователь может залогиниться
+ пока не подтвердит почту, логин не проходит
